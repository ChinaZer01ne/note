# 个人知识库

> 一个长期维护的学习与研究笔记站点，聚焦后端工程、系统设计、工程治理，以及经济、历史、哲学等延伸主题。

这里沉淀的是个人在学习、复盘和实践过程中的结构化记录。内容以主题归档、问题导向和长期可检索为目标，适合作为日常查阅、专题复习和知识串联的入口。

## 站点内容

- **知识地图**：整理复习主线、学习计划和阅读顺序，帮助快速进入上下文。
- **技术专题**：覆盖 Java、JVM、Spring、数据库、中间件、微服务、系统设计、稳定性治理等核心主题。
- **人文与思考**：收录经济、历史、投资、哲学方向的基础笔记与个人思考。
- **实践记录**：保留部署、试验和日常项目类记录，方便回顾具体落地过程。

## 建议阅读方式

1. 初次访问，建议先浏览左侧导航，了解站点的主题划分与内容范围。
2. 有明确问题时，直接通过搜索或对应分类进入具体文档，按需查阅。
3. 若关注后端工程、系统设计与工程治理，可优先阅读 `Summary` 目录中的专题总结。

## 维护原则

- **结构优先**：尽量按主题、层次和场景组织内容，减少零散堆积。
- **结论导向**：优先记录概念边界、核心机制、常见问题和取舍思路。
- **持续修订**：已有笔记会随着学习和实践反复更新，不追求一次成稿。

## 使用说明

- 本站基于 Docsify 构建，适合静态部署与在线检索。[怎么做的？](Summary/Markdown%20文档做成在线网站.md)
- 部分日常记录仅作为个人索引使用，不保证完整公开展示。
- 涉及投资相关内容仅为个人学习笔记，不构成任何投资建议。
