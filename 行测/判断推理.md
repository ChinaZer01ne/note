
## 直言命题

* **所有是**的矛盾是**有些非**
* **所有非**的矛盾是**有些是**
* **有些非**的矛盾是**所有是**
* **有些是**的矛盾是**所有非**
* **某个是**的矛盾是**某个非**
* **某个非**的矛盾是**某个是**


所有是 => 有些是
所有是 => 某个是 => 有些是
所有非 => 有些非
所有非 => 某个非 => 有些非

> 有些不推出任何东西，但能确定矛盾。
> * **有些非**的矛盾是**所有是**
> * **有些是**的矛盾是**所有非**

## 联言命题

* **A且B**，全真则真，一假为假，**A且B**的矛盾是**非A或非B**

## 选言命题

* **A或B**，一真则真，全假为假，**A或B**的矛盾是**非A且非B**

> 至少一个 === 或

* 要么A，要么B：**二者必须只能选其一**，要么A要么B的矛盾是**(A且B)或(非A且非B)**

> 要么要么，必须选一个，也只能选一个
