
## 必然性推理
### 直言命题

* **所有是**的矛盾是**有些非**
* **所有非**的矛盾是**有些是**
* **有些非**的矛盾是**所有是**
* **有些是**的矛盾是**所有非**
* **某个是**的矛盾是**某个非**
* **某个非**的矛盾是**某个是**


所有是 => 有些是
所有是 => 某个是 => 有些是
所有非 => 有些非
所有非 => 某个非 => 有些非

> 有些不推出任何东西，但能确定矛盾。
> * **有些非**的矛盾是**所有是**
> * **有些是**的矛盾是**所有非**

### 联言命题

* **A且B**，全真则真，一假为假，**A且B**的矛盾是**非A或非B**

### 选言命题

* **A或B**，一真则真，全假为假，**A或B**的矛盾是**非A且非B**

> 至少一个 === 或

* 要么A，要么B：**二者必须只能选其一**，要么A要么B的矛盾是**(A且B)或(非A且非B)**

> 要么A要么B，必须选一个，也只能选一个，如果不选A就选B，如果不选B就选A
### 假言命题

充分 => 必要

* 前推后：
	* **如果**A，**那么（就）**B；A=>B，A是充分条件，B是必要条件
	* **只要**A，**就**B；A=>B，A是充分条件，B是必要条件
	* **若**A**则**B；A=>B，A是充分条件，B是必要条件
* 后推前：
	* **只有**A，**才**B；B=>A，B是充分条件，A是必要条件
	* **除非**A，**否则不**B；B=>A，B是充分条件，A是必要条件
	*  **A是B的前提**：B=>A，前提、基础、先决条件都是必要条件，有了结果才表明前提是一定成立的
* 除非A，否则B；非A=>B，非A充分条件，B是必要条件。否一推一
* 我语法标志，默认前推后，如己所不欲勿施于人

> 充分条件：足够的条件，有他就够
> 必要条件：造成结果，需要这个条件，没他不行

* **A=>B**的矛盾是，**A且非B**
* **A=>B**的等价命题是其逆否命题，**非B=>非A**

## 可能性推理

### 削弱加强型

削弱：
* 另有他因
* 否定此因
* 切断因果
* 因果倒置
加强
* 排除他因
* 肯定此因
* 建立因果

> 最优选项：
> * 直接 > 间接
> * 量大 > 量小
> * 本质 > 现象
> * 综合 > 单一
> * 必然 > 偶然
